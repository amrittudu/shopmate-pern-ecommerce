# 🛒 ShopMate – AI Integrated E-Commerce Platform

A full-stack AI-powered e-commerce web application built using React, Node.js, PostgreSQL, and Stripe, with Google Gemini AI integration for intelligent product recommendations.

---

## 🚀 Features

- User Authentication (Register / Login)
- Product Listing, Search & Product Details Page
- AI-Powered Recommendations using Google Gemini
- Shopping Cart & Checkout System
- Secure REST APIs with Express & PostgreSQL
- Admin Dashboard (Add, Update, Delete Products)
- Order Management & User Profiles
- Secure Payment Integration (Stripe)
- Responsive UI with React + Tailwind CSS
- Scalable Architecture (Frontend + Backend + Admin Dashboard)

---

## 🧰 Tech Stack

**Frontend**
- React.js
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Database**
- PostgreSQL

**Payment**
- Stripe

**AI Integration**
- Google Gemini API
